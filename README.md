# Práctica PaaS - Aplicación Flask

## 📋 Descripción
Aplicación web simple desarrollada con Python Flask para demostrar el despliegue en Render como parte de la práctica de modelos de servicio en la nube (PaaS).

## 📂 Archivos del proyecto
- `app.py` - Código principal de la aplicación Flask
- `requirements.txt` - Dependencias (Flask y Gunicorn)
- `Procfile` - Configuración para Render

## 🚀 Despliegue
Esta aplicación está configurada para desplegarse en Render como servicio PaaS.
